# quecompramos

Este repositorio contiene la base de una aplicación para llevar una lista de compras de supermercado que pueda mantenerse siempre a mano y compartirse entre varias personas.

## Idea central

- Lista simple y siempre disponible.
- Acceso fácil con usuarios o sesiones temporales.
- Colaboración entre varias personas sobre una misma lista.
- Foco en cargar y completar productos sin fricción.

## Lo que deben recordar los agentes

- El producto prioriza simplicidad.
- La sesión temporal es parte del concepto base.
- La lista está pensada para uso compartido.
- La documentación principal del proyecto vive en `README.md` y `AGENTS.md`.

## Criterio de trabajo

- No introducir complejidad innecesaria.
- Mantener coherencia entre visión de producto y documentación.
- Si se agregan nuevas decisiones importantes, documentarlas en ambos archivos de referencia.
